We're a group of people in South Eastern Wisconsin looking to better our future by learning to code.

Some of us are students attending local colleges, others are mid career professionals looking to change careers, some just like to build things and want to do it with code. As long as you are looking to learn how to code free code camp kenosha is the place for you. You'll develop real world applications in a self paced, online enviroment. You'll learn front end, back end, and project management, all of the tools you need to be a web developer. To graduate you'll build a real production level web app to be used to a member non profit in their day to day operations.

We want you to help you learn.
